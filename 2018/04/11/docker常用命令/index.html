docker常用命令

2018-04-11
docker

前言

  • Docker 可以让开发者打包他们的应用以及依赖包到一个轻量级、可移植的容器中，然后发布到任何流行的 Linux 机器上，也可以实现虚拟化。

  • 容器是完全使用沙箱机制，相互之间不会有任何接口（类似 iPhone 的 app）,更重要的是容器性能开销极低。

  • images： docker images
  • ps： docker ps
  • 导入镜像：docker pull mongo:3.4
  • build：docker build -t nodeApp .
  • run：docker run -d -p 4000:4000 nodeApp
  • 删除：docker rm nodeApp
  • 停止：docker stop nodeApp
  • 查看日志：docker logs 1f9702808d7b
  • 进入宿主机器：docker exec -it 1f9702808d7b /bin/bash
  • link： docker run –name sec_app -d -p 4000:4000 –link secmongo:mongo -d seckill
