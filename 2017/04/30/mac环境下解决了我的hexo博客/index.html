mac环境下解决了我的hexo博客

2017-04-30
hexo

前言

  • 终于换了笔记本了，以前的asus A550终于退休了，买了个macbook pro也开始了我的吃土生活😄😄😄，然后问题就来了，MacBook下怎么搭建hexo博客，才能继续写写博客，喝喝小茶～

第一个问题，又遇到了ssh的问题，换电脑了，那么ssh又需要重新配置了。

  • 本地生成ssh key密钥命令：ssh-keygen -t rsa -C “your github email”
  • 查看.ssh／目录，可以看到公钥密钥以生成，名称分别为id_rsa.pub和id_rsa
  • 终端执行命令查看公钥内容：cat ~/.ssh/id_rsa.pub
  • 将这一部分代码添加到GitHub账号的SSH Keys中去（别忘了在前面加上ssh-rsa）
  • 点击add ssh确认就行了，这样ssh-key就解决了

    第二个问题，当然是重新解决hexo项目了。

  • npm install hexo-cli -g
  • hexo init blog
  • cd blog
  • npm install
  • hexo s

    第三个问题，就是解决你的主题，还有一些config配置。

  • 重新选择主题，我还是喜欢minos这个主题
  • 修改_config.yml
  • hexo d -g 部署到你的github.io上

    第四个问题，发布文章那些命令

  • hexo new [layout] “post-name”发表文章
  • hexo new page “pagename”新建页面
  • hexo
