每天写一点东西，不知跬步，无以至千里。

  • 最近买了本书学习JavaScript数据结构和算法，感觉这本书写的很不错的，简单明了。市场上就应该多出一点这样的书籍，话说我当初学数据结构的时候，那个时候才学过C语言，然而，学校这群脑残的，给我们挑了一本C++版本的数据结构，那是怎么的一个懵逼啊！代码看不懂，书还厚！简单的问题复杂化。这就是我对数据结构实用教程（第二版）的总结。

总结几点学习经验

  • 重新理解了数组、栈、队列、链表、集合、树、图
  • 学习了几个算法知识（记得之前看过一个腾讯的一个题目50阶台阶的问题，一直存在疑问，看了斐波拉契数列，这个问题迎刃而解了。）
  • 如果你在使用JavaScript这门语言的话，对算法和数据结构有疑问的，建议可以推荐看这本书。
