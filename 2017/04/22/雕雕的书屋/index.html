雕雕的书屋

2017-04-22
node.js

前言

  • 好久没有写点技术性的项目文章了，历经好几个月打造的雕雕的书屋终于算是完成了。
  • 这个项目有参考YouTube的fast record这个兄弟的小视频。（不过这位兄弟账号被YouTube官方给封了，说什么欺骗行为什么的）
  • YouTube真是很不给面子，卡就算了，还让我开发这个项目开发到了一半多一点的时候，把人家的账号给封了。[哈哈]

所需要的环境

  • node.js
  • npm
  • mongoDB
  • 当然必不可少的是编辑器（sublime,vscode等等等）

    功能

  1. 后台管理
  2. 添加购物车
  3. 通过管理员来添加商品分类还有商品信息
  4. 商品详情

    项目实现后截图

  5. 主页
  6. 关于
  7. 购物车
  8. 详情页
  9. 书目列表
  10. 添加书籍
  11. 分类页
  12. 添加分类

    怎么启动该项目

  • git clone git://github.com/AlfieriChou/bookStore.git
  • npm install
  • npm start

    说明

  • 这个项目是基于PayPal所开发的kraken.js所完成的，前端采用foundation框架，为什么没用bootstrap，个人感觉foundation还简单一点代码量较bootstrap少一点。所以就选择了foundation。
  • 虽然采用了比较少见的框架来开发。但是对于你的开发能力也是一种挑战，应为在开发过程中，不确定因素太多。本来我是打算用阿里的egg框架来开发的，但是教程相对来说较少，而且，在国内的话，接入支付宝会很方便，谁会那么煞笔的去使用PayPal，除非是外贸。
  • 这个项目我以后会更新，应为这个系统还没有添加管理员账号和用户登录注册还有用户管理功能。我以后会继续完善这个系统。
