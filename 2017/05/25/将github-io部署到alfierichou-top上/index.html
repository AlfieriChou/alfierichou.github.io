将github.io部署到alfierichou.top上

2017-05-25
cname

前言

  • 本来想买个.com的域名，因为主要是为了练手，所以就买了个便宜的.top域名。

步骤

  • 前提是你已经开通了github.io博客
  • 在你的github.io新建一个CNAME的项目
  • 在CNAME下填写你的网址，比如说我的就是alfierichou.top前面带的http://就不要填了
  • 在你的域名解析填写三条信息
  • A www 默认 192.30.252.153
  • CNAME @ 默认 alfierichou.github.io
  • A www 默认 192.30.252.154
  • 浏览器打开alfierichou.top即可
