对JavaScript操作数组的研究

2017-07-05
javascript

array数组

  • concat()——连接两个或更多的数组，并返回结果
    具体使用——先定义一个数组，然在通过arr.concat(arr2)实现两个数组的拼接

  • join()——把数组的所有元素放入一个字符串。元素通过指定的分隔符来进行分割
    具体使用——arr.join(“.”)这是用.来分割字符串中的每一个元素

  • pop()——删除数组中的最后一个元素
    具体使用——arr.pop()将数组中放入最后一个元素删除（可以用数据结构中的栈来理解-先进后出）

  • push()——向数组的末尾添加一个或者多个元素，并返回新的长度
    具体使用——arr.push(“jamas”)这是向数组中添加一个jamas元素（这个功能可以用添加一段HTML代码这是经常使用的）这个也可以用数据结构中的栈来理解

  • reverse()——颠倒数组中元素的顺序
    具体使用——arr.reverse()将数组中的元素倒序排列

  • shift()——将数组中的第一个元素删除，并返回第一个元素的值
    具体使用——arr.shift() （删除最后一个是采用pop()）

  • slice()——可以从已有的数组中返回选定的元素（重点）
    具体使用——arr.slice(start,end)

  • toSource()——表示对象写源代码（这个只支持gecko浏览器Firefox，了解一下就行了）

  • toString()——可以将数组转换成字符串，并返回结果。
    具体使用——arr.toString()

  • toLocaleString()——把数组转换为本地字符串
    具体使用——arr.toLocaleString()
    {toString和toLocaleString有什么区别，我也不知道，不过自己在用的时候一般都是在用的toString}

  • unshift()——向数组的开头添加一个或更多的元素，并返回新的长度。（无法再IE中使用）
    具体使用——arr.unshift()

  • valueOf()——返回array对象的原始值。
    具体使用——arr.valueOf()
